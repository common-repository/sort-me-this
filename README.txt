Manage your WordPress media in a deeper and more precise way!

== Description ==

As the number of media and attachments increases, it can become difficult to find the file you are looking for.
SortMeThis allows to manage your WordPress media in a deeper and more precise way.
With this plugin you can filter the results by time period, file type or by Media Categories. Media Categories are a sort of "tag" you can add to Media Files to identify it as part of a group.
Using Media Categories will increase the filters parameters functionality and will let you save time searching.
Users can add, edit or delete Media Categories directly from SortMeThis admin panel.
Last but not least, you can view media information, edit or delete the media directly from the SortMeThis administration page.

Premium version features:
- Unlimited Media Categories
- Up to 100 display Media per page
- Add Media Category to multiple selected Medias
- Customer support



== Installation ==

1. Upload 'sort-me-this' folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In admin menu just click on SortMeThis to view plugin pages

== Frequently Asked Questions ==

= How many  Media Categories can I create? =

In the free version you can create up to 5 Media Categories.
In Premium version you are allowed to create as many as you want :)

= Can I add Media Categories to existing media? =

Sure you can add Media Categories to already uploaded media.

= How many Media Categories can I add to a media file? =

You are free to add as many as you want Media Categories to your media file.
Keep in mind that you can only have up to 5 Media Categories in free version.
